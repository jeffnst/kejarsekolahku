Nama

API login siswa

URL

http://localhost/pkbmksi-web/api/v1/siswa/login

Method        

POST

Tipe Data

JSON

Parameter

idsiswa & password (md5)

Contoh Parameter

{ "idsiswa": "KSI00115001","password": "25d55ad283aa400af464c76d713c07ad" }

Response

JWT Token dengan payload data berisi response , role , dan data .

Contoh response

eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.

eyJyZXNwb25zZSI6Ik9LIiwicm9sZSI6IjMiLCJkYXRhIjp7ImlkX3Npc3dhIjoiS1NJMDAxMTUwMDEifX0.

hUdpfJ2LoouM5-40pvDRdqTDYrgZS2M18u9GGF58-8

Response setelah decode        

{  "typ": "JWT",

 "alg": "HS256"

}{

 "response": "OK",

 "role": "3",

 "data": {    

            "id_siswa": "KSI00115001"  

            }

}

Logic Server        

1. Terima input JSON dari client berisi idsiswa dan password

2. Decode JSON yang telah diterima

3. Tampung idsiswa dan password dalam array.

4. Parsing data array ke model untuk query di database untuk di validasi

5. Jika validasi benar, maka bentuk array berisi response,role,dan data

6. Encode Json berisi array tadi dengan JWT agar lebih aman.
